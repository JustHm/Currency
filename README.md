# Currency APP [IOS]
## 프로젝트 설명
- 나라별 한국과의 환율을 표시해주고 계산할 수 있는 어플

## 화면 구성

- 환율 표시 리스트 **(첫 화면) + 위젯**
- 환율 정보 추가 화면
- 국가 선택 화면
